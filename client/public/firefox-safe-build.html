Firefox Build Issue Detected
Firefox has compatibility issues with the build process due to Service Worker conflicts in version 141.0+

Recommended Solutions:

  • Switch Browser: Use Chrome, Edge, or Safari for building
  • Development Mode: Continue using Firefox for development
